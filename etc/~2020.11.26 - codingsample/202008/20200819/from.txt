areSimilar
입력으로 받은 두 정수배열이 비슷한지를 출력하는 함수.
두 정수배열은 완전히 같거나, 한쌍의 위치를 서로 바꿨을때, 일치한다면 비슷한걸로 침.

아이디어
일단 모든 요소에 대해 검증해야 하기 때문에 배열크기만큼 반복을 해야함.
두 배열의 크기가 다를경우 false(일치할 수 없음)
하나씩 일치한지 확인해감.
일치하지 않을시, 해당 값과 배열 위치를 별도로 저장.
	저장하지 않고 할 수 있는 방법이 있을까?
	=>해당 위치만 저장하면 배열값을 통해 값을 가져올 수 있음.

a와 b 배열에서

1) if 두 배열의 크기가 다를경우 : false
2) 반복문 [ i는 0부터 a의 크기까지 ]:
	2-1) a[i]와 b[i]가 일치한지 체크
	2-2) if 두 값이 다를경우:
		2-2-1) if errpoint가 -1이 아니라면(이미 다른값이 나온경우)
			2-2-1-1) if errpoint가 -2라면(이미 스왑이 이루어진경우) false
			2-2-1-2) else (스왑이 이루어지기 전) -> a[errpoint]와 b[i], a[i]와 b[errpoint]가 서로 일치한지 확인
				일치할 경우 errpoint=-2, 일치하지 않으면 false
		2-2-2) else errpoint=i (errpoint를 초기에 -1로 초기화를 시켜두어야함)
3) 스와핑포인트만 설정되고 스왑핑이 이루어지지 않을경우 : false(errpoint>-1)
4) 스와핑이 정상적으로 이루어진경우(else) : true




arrayChange
정수배열이 순수 증가수열이 되기 위해서 올려야 하는 수의 총합을 반환

아이디어
모든 배열을 1부터 끝까지 반복하며
a[i]<=a[i-1]일때, a[i-1]-a[i]+1 을 총합 sum에 더해줌.
이를 끝까지 반복하면 끝

inputArray 에서
1) if 배열의 크기가 1이라면 0반환
2) 반복문 [ i는 1부터 inputArray의 크기까지 ]:
	2-1) if a[i] <= a[i-1] 일때, sum+=a[i-1]-a[i]+1
		그리고 a[i] = a[i-1]+1로 설정.
3) return sum
