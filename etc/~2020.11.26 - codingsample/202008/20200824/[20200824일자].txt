
1번문제
arrayReplace
정수 배열이 주어지고, 배열 내에 존재하는
elemToReplace 요소를 substitutionElem으로 변경하는 함수

내아이디어 :
배열 전체를 반복해야할 것으로 보이고,
연산 자체는 
1) 배열 전체를 도는 회수(배열의 크기만큼 연산)
2) 특정 값을 찾았을 때, 해당 값을 substitutionElem으로 변경하는 연산
=> 크기 + elemToReplace의 개수

C언어
위의 방법보다 더 나은 방법은 찾기 힘듦.

파이썬
1) 같은 방법으로 구현이 가능함.
2) 짧게 구현을 위해서, 한줄로 적음
한줄로 리스트 요소를 적용할때, 요소 조건을 붙이는 것에서
i if else 후에 for문을 적음으로 이것이 가능해진다.
elif가 가능한지는 아직 모르겠음.

자바
1) 같은 방법으로 구현이 가능함.
2) 스트림을 이용하여 파이썬과 같이 쉽게 구현이 가능함.
map()을 사용할때, 배열의 조건을 붙이는 방법으로
i -> i==조건 ? 해당값 : i 로 i를 변경해줄 수 있음.

2번문제
evenDigitsOnly
정수로 주어진 수가 모두 짝수로만 이루어져있는지 체크하는 함수.

내 아이디어:
temp 변수를 하나 두면서
temp값에는 주어진 정수 n의 한자리씩 읽어가는 함수임.
=> 10씩 나눠주는 방법, 문자열로 변경하는 방법(C언어는 힘들것임)

1번방법 10씩 나눠주기
1) temp에 n을 적용
2) temp를 10 mod연산으로 1의자리를 찾아냄.
3) 해당 1의자리가 짝수인지 판정 아니면 false
4) n을 10으로 나눠주어 한칸 앞으로 전진 (이를 반복)
5) 모두 반복했을때(n!=0이 아닐때) true

2번방법 문자열로 변경하기
1) n을 string으로 변경 (output)
2) output의 길이만큼 끝에서 첫째자리까지 읽음
3) 각 읽은 값을 정수로 다시 변경했을 때, mod 2가 0이 아니면 false

C언어
크게 다른방법이 없음

자바
1) 스트림을 이용함
해당 정수값을 String 배열로 만들어서
개당값을 Allmatches로 i % 2 == 0인지를 파악하면 됨.
2) matches 함수를 이용
2-1) (정수+"") 하면 스트링이됨!!

파이썬
1) all을 이용해, n을 str으로 만들어,
해당 스트링을 돌면서 각 값이 %2==0인지 체크 나머지는 다 비슷한듯


3번문제
variableName
변수이름 조건에 주어진 문자열이 맞는지를 확인.
해당 문제는 함수이용이 중요하므로,
여러가지 사용할 수 있는 함수들에 대해서 알아봐야함.

C언어
isalpha(해당 char값이 알파벳인지를 확인)
isdigit(해당 char값이 숫자인지를 확인)
isalnum(해당 char값이 알파벳이나 숫자인지를 확인)

파이썬
isidentifier(자체적으로 식별자(변수이름)인지를 확인하는 함수)

자바
마찬가지로 matches함수를 이용
[][]* 괄호 두개를 이용하면
첫번째는 첫번째 요소부터 끝까지,
두번째는 두번쨰 요소부터 끝까지에 적용됨


4번문제
alphabeticShift
문자열이 주어지면, 해당 문자열 알파벳의 다음에 해당하는 알파벳으로 변환하는 함수.

나의 아이디어:
+1하면 되겠죠?

파이썬
ord사용해서 ASCII로 만든다음에 +1해주고, 해당값을 다시 char형으로 바꾸어서
해당 문자열의 위치로 적용해주면 될듯 합니다.
ord <-> chr함수 이용하면 될것같음.
